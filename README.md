# S86-1225-AbyssWatchers-NextjsAnd-AWS-Azure-SafeShore_1

## **Project Overview :**

- The goal of the project is for helping people during floods and such calamities. This Project's Idea is to notify and alert people when floods occur.

### **Tech Stack :**

- NextJs
- AWS
- Tailwindcss
- Docker
- Prisma

- **Additional Info:** _Using Tailwindcss, Typescript, node environemnet_

**Key Ideas:**

- Get Data from weather apis where they have lot of data and use this for prediction.
- Use an Algorithm or prediction logic system which uses the current data like from weather api, the location's past prone chances, other external  
  factors like rate of soil errosion.
